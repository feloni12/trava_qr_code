Olá, para instalar sua fechadura, é necessário seguir esses passos:

1- procure pela dashboard, ou aperte Ctrl + Alt + T

2- No dashboard, escreva o comando 

	chmod +x setup.py

3- Execute o setup com o comando 

	./setup.py

3- Abra o inicial_ com o comando

	python inicial_.py

4- Abra o inicial_2 com o comando

	python inicial_2.py

Depois disso, seu sistema estára rodando e pronto para funcionar.

!!!!Lembre-se de abrir o setup.py primeiro, caso contrário os scripts inicial_ e inicial_2 não funcionarão!!!!
